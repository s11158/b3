# b3.gg — сокращалка ссылок (Cloudflare Worker + KV)

Публичный сервис коротких ссылок на `b3.gg/<код>`. Главный сайт остаётся на
GitHub Pages (репозиторий `s11158/b3`), а Worker добавляет слой сокращалки и
проксирует всё остальное на Pages.

## Как работает

- `POST /api/shorten` `{ "url": "..." }` → `{ code, short, url }`
- `GET /api/stats` → сколько кодов выдано, режим (выдача / переработка)
- `GET /<код>` → `302`-редирект на длинный URL
- остальное → проксируется на `https://s11158.github.io/b3`

Коды: алфавит `0-9 a-z A-Z` (62). Сначала 1-символьные (62 шт), затем
2-символьные (3844 шт) — всего пул **3906**. Когда пул исчерпан, слоты
переиспользуются по кругу, начиная с самого старого (FIFO). Бесконечно.

## Развёртывание

Требуется Node.js и аккаунт Cloudflare с добавленным доменом `b3.gg`.

```bash
cd worker
npm i -g wrangler            # или npx wrangler ...
wrangler login               # авторизация в Cloudflare

# 1. создать KV-неймспейс и вставить выданный id в wrangler.toml
wrangler kv namespace create LINKS

# 2. задеплоить Worker (создаст и привяжет маршрут b3.gg/*)
wrangler deploy
```

### Предусловия в Cloudflare (один раз)

1. Домен `b3.gg` добавлен в Cloudflare, неймсерверы Porkbun переключены на
   выданные Cloudflare.
2. DNS: записи на GitHub Pages (A `185.199.108–111.153`), оранжевое облако
   (proxied). Запись `www` — CNAME на `s11158.github.io`, тоже proxied.
3. SSL/TLS mode = **Full**.
4. В GitHub Pages репозитория `s11158/b3` **снять кастомный домен** (удалить
   файл `CNAME`, очистить cname), иначе `s11158.github.io/b3` редиректит на
   `b3.gg` и Worker уходит в петлю.

## Локальный тест

```bash
wrangler dev          # поднимет воркер локально с превью KV
```

## Настройки (в worker.js)

- `MAX_LEN` — макс. длина кода (по умолчанию 2).
- `ALPHABET` — набор символов.
- `RATE_LIMIT_PER_DAY` — лимит создания ссылок на IP/сутки.
- `RESERVED` — пути, которые не считаются кодом.

## Ограничения v1

- KV не атомарен: при очень высокой одновременной нагрузке два запроса могут
  получить один слот. Для личного/умеренного трафика не проблема. Апгрейд —
  Durable Objects для атомарного счётчика.
- Бесплатный KV: 100k чтений/сутки, 1k записей/сутки. Редиректы — только
  чтение (быстро), запись лишь при создании ссылки.
